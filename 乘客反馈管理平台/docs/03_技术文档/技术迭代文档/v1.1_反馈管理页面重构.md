## 3. 数据流设计

```
FilterBar (FilterStore)
       ↓
FeedbackManagement 页面
       │
       ├── 模块1：筛选条件 → FilterBar 组件
       │
       ├── 模块2：反馈列表 → useFeedbacks + filters
       │
       ├── 模块3：数据仪表盘 → useOverviewStats + useTrendData + useDistributionData + filters
       │
       └── 模块4：AI智能分析 → useAISummary + filters, useAISuggestions + filters
```

## 4. 页面内导航

使用页面内滚动 + 二级菜单高亮联动：

```typescript
// 页面结构
<div>
  <section id="list">反馈列表内容</section>
  <section id="dashboard">数据仪表盘内容</section>
  <section id="ai">AI智能分析内容</section>
</div>

// 导航点击处理
const scrollToSection = (id: string) => {
  document.getElementById(id)?.scrollIntoView({ behavior: 'smooth' })
  setActiveSection(id)
}
```
