## 3. 数据流变更

### 3.1 AI 分析筛选参数传递

```
用户选择"当前筛选"
       ↓
useFilterStore 获取全局筛选条件
       ↓
AiAnalysis.tsx 组装 API 参数
       ↓
传递给 useAISummary / useAISuggestions
       ↓
POST /api/ai/summary 或 /api/ai/suggestions
       ↓
AIService._get_feedback_data_for_analysis 应用筛选条件
       ↓
返回筛选后的数据给 AI 进行分析
```

### 3.2 问题分类饼图数据来源

```
变更前：suggestions（AI 生成的建议列表）
变更后：type_distribution（实际的问题类型统计）
```

## 5. 核心代码变更

### 5.1 AIService._get_feedback_data_for_analysis

新增筛选条件处理：
```python
if rating_min is not None:
    conditions.append(Feedback.rating >= rating_min)

if rating_max is not None:
    conditions.append(Feedback.rating <= rating_max)

if status and len(status) > 0:
    conditions.append(Feedback.status.in_(status))

if feedback_type and len(feedback_type) > 0:
    # JSON 数组字段，在 Python 层过滤
    pass

if keyword:
    conditions.append(Feedback.feedback_text.contains(keyword))
```
