# v1.5 测试迭代文档 - AI智能分析模块重构

**版本**: v1.5
**日期**: 2026-04-17
**状态**: 待评审

---

## 1. 概述

### 1.1 测试背景

v1.5 版本对 AI 智能分析模块进行重构，实现完整的多步骤分析流水线。

### 1.2 测试范围

- 分析入口测试（点击"开始分析"按钮）
- 数据筛选测试（基于 FilterBar 筛选条件）
- 数据规模测试（默认100条，最多2000条）
- 分析流水线测试（数据清洗 → 摘要 → 问题分析 → 建议）
- 异步任务测试（任务创建、轮询、完成/失败）
- 结果展示测试（摘要、问题分类、优化建议三块结果）

---

## 2. 功能测试

### 2.1 分析入口测试

| 测试用例 | 操作 | 预期结果 |
|----------|------|----------|
| 首次点击 | 点击"开始分析"按钮 | 发起分析请求，按钮变为"分析中..." |
| 二次点击 | 分析完成后点击"重新分析" | 重新发起分析 |
| 空数据筛选 | 无筛选条件点击分析 | 使用默认100条数据 |

### 2.2 数据筛选测试

| 测试用例 | 操作 | 预期结果 |
|----------|------|----------|
| 城市筛选 | 设置城市筛选后点击分析 | 只分析该城市数据 |
| 时间范围筛选 | 设置时间范围后点击分析 | 只分析该时间范围内数据 |
| 评分筛选 | 设置评分范围后点击分析 | 只分析符合评分条件数据 |
| 多条件组合 | 设置多个筛选条件后点击分析 | 只分析符合所有条件数据 |

### 2.3 数据规模测试

| 测试用例 | 筛选结果 | 预期分析条数 |
|----------|----------|--------------|
| 无筛选 | - | 100 条 |
| 筛选 > 2000 条 | 5000 条 | 2000 条 |
| 筛选 ≤ 2000 条 | 1500 条 | 1500 条 |

## 3. 分析流水线测试

### 3.1 数据获取测试

| 测试用例 | 操作 | 预期结果 |
|----------|------|----------|
| 正常获取 | 发起分析 | 正确获取筛选后的数据 |
| 无数据 | 筛选无结果 | 返回空数据提示 |
| 部分失败 | 数据获取部分失败 | 继续处理可用数据 |

### 3.2 数据清洗测试

| 测试用例 | 输入 | 预期输出 |
|----------|------|----------|
| 正常数据 | 有效反馈列表 | 不变 |
| 空文本过滤 | 包含空文本的反馈 | 过滤空文本 |
| 纯表情过滤 | 纯表情反馈 | 过滤纯表情 |
| 排序验证 | 反馈列表 | 低分在前，高分在后 |

### 3.3 摘要生成测试

| 测试用例 | 操作 | 预期结果 |
|----------|------|----------|
| 正常生成 | 分析完成 | 生成150字以内摘要 |
| 好评为主 | 筛选好评数据 | 摘要以正面为主 |
| 差评为主 | 筛选差评数据 | 摘要以负面为主 |

### 3.4 问题分析测试

| 测试用例 | 操作 | 预期结果 |
|----------|------|----------|
| 正常分类 | 分析完成 | 反馈归类到已有分类 |
| 新问题发现 | 反馈包含新问题模式 | AI 发现并命名新分类 |
| 多分类 | 单条反馈多种类型 | 支持多分类 |

### 3.5 建议生成测试

| 测试用例 | 操作 | 预期结果 |
|----------|------|----------|
| 正常生成 | 分析完成 | 生成可落地建议 |
| 用户原声 | 建议生成 | 引用用户原声 |
| 优先级 | 多个建议 | 按严重程度排序 |

## 7. 验收标准

### 7.1 功能验收

- [ ] 点击"开始分析"按钮后发起分析请求
- [ ] 分析使用顶部 FilterBar 的筛选条件
- [ ] 无筛选时默认分析最新 100 条数据
- [ ] 筛选结果 > 2000 条时只分析 2000 条
- [ ] 筛选结果 ≤ 2000 条时分析全部数据
- [ ] 分析完成后展示三块结果：摘要、问题分类、优化建议
- [ ] 支持"重新分析"操作

### 7.2 UI 验收

- [ ] 标题、数据预览、按钮在同一行
- [ ] 按钮 loading 状态正确
- [ ] 卡片布局符合响应式设计
- [ ] 优先级颜色正确显示

### 7.3 错误处理验收

- [ ] 网络错误时显示错误提示
- [ ] 服务异常时显示异常提示
- [ ] 重试操作正常
