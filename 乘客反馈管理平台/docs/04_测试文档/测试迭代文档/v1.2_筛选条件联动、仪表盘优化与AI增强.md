**版本**: v1.2

**状态**: 已完成

## 1. 测试概述

本文档描述 v1.2 版本中筛选条件联动与AI增强功能的测试计划。

---

## 2. 测试范围

### 2.1 功能测试

| 模块 | 测试内容 |
|------|---------|
| 全局筛选状态 | useFilterStore 状态读写、筛选条件组合 |
| Dashboard 联动 | 指标卡片、趋势图、分布图响应筛选变化 |
| AI分析联动 | "当前筛选"选项功能验证 |
| 后端 API | 各筛选参数的正确性和组合逻辑 |

### 2.2 回归测试

| 模块 | 测试内容 |
|------|---------|
| 反馈列表 | 原有筛选功能不受影响 |
| Dashboard | 时间范围切换（向后兼容） |
| AI分析 | "全部反馈"和"指定时间"选项不受影响 |

## 6. 测试数据准备

### 6.1 筛选条件测试数据

| 筛选类型 | 测试数据 |
|----------|---------|
| 城市 | 深圳(多条)、北京(多条)、广州(多条) |
| 评分 | 1星、3星、5星 各多条 |
| 状态 | 待处理、处理中、已处理、已关闭 各多条 |
| 反馈类型 | 行驶体验、车内环境、事故响应 等各多条 |
| 关键词 | 包含"急刹车"、"空调"、"事故"等 各多条 |

### 6.2 组合筛选测试数据

- 城市=深圳 AND 评分≥4
- 状态=待处理 AND 反馈类型=行驶体验
- 城市=北京 OR 广州 AND 状态=已处理
