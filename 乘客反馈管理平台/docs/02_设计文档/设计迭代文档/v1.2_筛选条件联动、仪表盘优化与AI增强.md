## 3. 图表增强设计

### 3.1 路线分布柱状图

**位置**：数据仪表盘，城市分布柱状图右侧

**图表规范**：

| 属性 | 规范值 |
|------|--------|
| 类型 | 柱状图（bar） |
| X轴 | 路线名称 |
| Y轴 | 反馈数量 |
| 柱状颜色 | 主色调 #FF6033 |
| 柱状圆角 | 上方圆角 4px |
| 柱子宽度 | 60% |

### 3.2 时段热力图

**位置**：数据仪表盘，独立一行

**图表规范**：

| 属性 | 规范值 |
|------|--------|
| 类型 | 热力图（heatmap） |
| X轴 | 小时（0-23） |
| Y轴 | 固定维度 |
| 颜色渐变 | 浅色 → 深色 |
| 配色方案 | #FFF7F0 → #FF6033 |

**热力图颜色规范**：
```
value=0   → #FFF7F0 (极浅橙)
value=低  → #FFDCC2 (浅橙)
value=中  → #FFAB66 (中橙)
value=高  → #FF6033 (深橙)
value=极高 → #CC3300 (极深橙)
```
