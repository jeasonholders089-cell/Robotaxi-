### 2.2 左侧导航设计

**一级菜单**（始终可见）：
- 工作台：icon + 文字
- 用户反馈：icon + 文字 + 展开箭头

**二级菜单**（用户反馈展开后，缩进显示）：
- 反馈列表
- 数据仪表盘
- AI智能分析

**高亮状态**：
- 一级菜单：选中时左侧有橙色指示条 + 文字变橙色
- 二级菜单：选中时文字变橙色（仅选中状态，无 hover 高亮）

### 2.3 右侧内容区设计

#### 模块一：筛选条件
- 标题：筛选条件
- 右侧有"展开筛选"链接

#### 模块二：反馈列表
- 标题：反馈列表
- 包含统计栏（总条数、已选条数、批量操作按钮）
- FeedbackTable 表格
- 分页控件

#### 模块三：数据仪表盘
- 标题：数据仪表盘
- 时间范围切换器（今日、本周、本月、自定义）
- 4个指标卡片：总反馈数、平均评分、好评率、差评率
- 反馈数量趋势折线图
- 评分分布饼图
- 路线分布柱状图
- 时段分布热力图
- 反馈类型分布横向柱状图

#### 模块四：AI智能分析
- 标题：AI智能分析
- AI摘要 + 重新生成/复制按钮
- 问题分类分布饼图
- 产品优化建议卡片

## 3. 交互设计

### 3.1 导航交互

1. 点击"用户反馈"一级菜单：展开/收起二级菜单（带箭头旋转动画）
2. 点击二级菜单：
   - 高亮该菜单项
   - 页面滚动到对应模块
3. 点击其他一级菜单（如"工作台"）：切换到对应页面

### 3.2 筛选联动

- 筛选条件变更时，页面内三个模块（反馈列表、数据仪表盘、AI智能分析）同步更新
- 使用 FilterStore 统一管理筛选状态

---

## 4. 验收标准

- [ ] 左侧导航有一级菜单：工作台、用户反馈
- [ ] 用户反馈下有二级菜单：反馈列表、数据仪表盘、AI智能分析
- [ ] 二级菜单选中高亮，非选中态无高亮
- [ ] 右侧四个模块（筛选条件、反馈列表、数据仪表盘、AI智能分析）有明确标题，按从上到下顺序排列
- [ ] 点击二级菜单，页面滚动到对应模块
- [ ] 筛选条件变更，三模块数据同步
