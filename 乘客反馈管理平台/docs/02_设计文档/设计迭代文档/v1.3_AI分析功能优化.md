## 1. 概述

## 2. 页面交互变更

### 2.1 AI 分析页面筛选联动

**变更前**（筛选联动无效）：
```
┌─────────────────────────────────────────────────────────────┐
│  分析范围：○ 全部反馈  ○ 当前筛选  ○ 指定时间              │
├─────────────────────────────────────────────────────────────┤
│  当前筛选选项 → 未传递任何筛选参数给 API                    │
└─────────────────────────────────────────────────────────────┘
```

**变更后**（筛选联动生效）：
```
┌─────────────────────────────────────────────────────────────┐
│  分析范围：○ 全部反馈  ○ 当前筛选  ○ 指定时间              │
├─────────────────────────────────────────────────────────────┤
│  当前筛选选项 → 传递以下参数给 API：                       │
│    - start_date, end_date                                  │
│    - city, rating_min, rating_max                          │
│    - status[], feedback_type[]                            │
│    - keyword                                               │
└─────────────────────────────────────────────────────────────┘
```

### 2.2 问题分类饼图数据来源

**变更前**：
```
问题分类分布饼图 → 使用 suggestions（AI建议数据）❌
```

**变更后**：
```
问题分类分布饼图 → 使用 type_distribution（实际分类分布）✅
```

## 3. 组件状态变更

### 3.1 useAISummary Hook

新增参数：
```typescript
interface AISummaryParams {
  start_date?: string
  end_date?: string
  city?: string
  rating_min?: number      // 新增
  rating_max?: number      // 新增
  status?: string[]        // 新增
  feedback_type?: string[]  // 新增
  keyword?: string          // 新增
  length?: 'short' | 'medium' | 'long'
  max_count?: number
}
```

### 3.2 useAISuggestions Hook

新增参数：
```typescript
interface AISuggestionsParams {
  start_date?: string
  end_date?: string
  city?: string
  rating_min?: number      // 新增
  rating_max?: number      // 新增
  status?: string[]        // 新增
  feedback_type?: string[]  // 新增
  keyword?: string          // 新增
  top_n?: number
}
```
