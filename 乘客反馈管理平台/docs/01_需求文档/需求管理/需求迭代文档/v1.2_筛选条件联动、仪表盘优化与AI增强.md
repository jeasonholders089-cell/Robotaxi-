# v1.2 需求迭代文档 - 筛选条件联动、仪表盘优化与AI增强

**版本**: v1.2
**日期**: 2026-04-17
**状态**: 已完成

---

## 1. 概述

### 1.1 背景

当前系统存在以下问题：

**问题一：Dashboard统计与筛选条件脱节**
- 数据仪表盘只根据时间范围获取统计数据
- 无法针对筛选条件（城市、评分、状态、反馈类型等）进行精准统计

**问题二：AI智能分析"当前筛选"功能无效**
- 选择"当前筛选"时，实际并未将筛选条件传递给后端

**问题三：数据仪表盘图表缺失**
- 原始设计要求路线分布柱状图，当前未实现
- 时段分布应为热力图，当前为横向柱状图

### 1.2 目标

1. 实现筛选条件全局共享，Dashboard 和 AI分析 响应筛选变化
2. 新增路线分布柱状图
3. 将时段分布改为热力图

---

## 2. 功能需求

### 2.1 全局筛选状态管理

- 利用 `useFilterStore` 作为全局筛选状态
- Dashboard 和 AI分析 订阅全局状态
- 反馈列表页面修改筛选条件时，同步更新全局状态

### 2.2 Dashboard 筛选联动

支持的筛选参数：
- 时间范围（已有）
- 城市（已有）
- 评分范围（新增）
- 反馈类型（新增）
- 状态（新增）
- 关键词（新增）

### 2.3 AI智能分析筛选联动

- 扩展后端 AI分析接口支持筛选参数
- "当前筛选"选项实际生效

### 2.4 数据仪表盘图表增强

#### 2.4.1 路线分布柱状图（新增）

**位置**：数据仪表盘

**图表要求**：
- 类型：柱状图（bar chart）
- 展示内容：各路线反馈数量分布
- 数据来源：`/stats/distribution` 接口的 `route_distribution` 字段
- 展示数量：Top 10

**设计示意**：
```
┌────────────────────────────────┐
│  路线分布（柱状图）            │
│                                │
│  路线A ████████████  45%     │
│  路线B █████████     28%     │
│  路线C ██████       18%     │
│  ...                          │
└────────────────────────────────┘
```

#### 2.4.2 时段分布热力图（改造）

**位置**：数据仪表盘，替换现有的横向柱状图

**图表要求**：
- 类型：热力图（heatmap）
- 展示内容：24小时各时段反馈数量密度
- 数据来源：`/stats/distribution` 接口的 `hour_distribution` 字段
- X轴：小时（0-23）
- 颜色渐变：浅色（低）→ 深色（高）

**设计示意**：
```
┌────────────────────────────────┐
│  时段分布（热力图）            │
│                                │
│  0  3  6  9  12 15 18 21 23  │
│ ┌────────────────────────────┐ │
│ │░░▓▓▓░░▓▓▓▓▓░░░▓▓▓▓░░░▓▓│ │
│ └────────────────────────────┘ │
│       24小时反馈密度           │
└────────────────────────────────┘
```
