**版本**: v1.1
**日期**: 2026-04-17
**状态**: 待开始

---

## 1. 概述

### 1.1 背景

当前 Dashboard、FeedbackList、AI Analysis 三个模块各自独立，页面割裂。用户需要在不同页面间跳转，筛选条件也要重复设置，流程不连贯。

### 1.2 目标

将三个功能模块整合到同一个页面，统一筛选条件，左侧导航改为二级导航结构。

### 2.2 左侧导航结构

**一级菜单**（始终可见）：
- 工作台
- 用户反馈（可展开）

**二级菜单**（用户反馈展开后）：
- 反馈列表（默认）
- 数据仪表盘
- AI智能分析

**导航行为**：
- 点击"用户反馈"一级菜单：展开/收起二级菜单
- 点击二级菜单：在右侧内容区滚动到对应模块，并高亮该二级菜单
- 二级菜单只在被选中时高亮（非 hover 高亮）

### 2.3 右侧内容区模块

每个模块有明确的标题名称，从上到下顺序为：

#### 2.3.1 筛选条件
- 标题：筛选条件
- 复用现有 FilterBar 组件
- 支持：评分范围、城市、反馈类型、状态、关键词搜索

#### 2.3.2 反馈列表
- 标题：反馈列表
- FeedbackTable 表格展示
- 包含统计栏（总条数、已选条数、批量操作按钮）
- 支持分页、排序、批量操作

#### 2.3.3 数据仪表盘
- 标题：数据仪表盘
- 时间范围切换器（今日、本周、本月、自定义）
- 4个指标卡片：总反馈数、平均评分、好评率、差评率
- 反馈数量趋势折线图
- 评分分布饼图
- 路线分布柱状图
- 时段分布热力图
- 反馈类型分布横向柱状图

#### 2.3.4 AI智能分析
- 标题：AI智能分析
- AI摘要卡片（重新生成、复制按钮）
- 问题分类分布饼图
- 产品优化建议列表

### 2.4 筛选条件联动

- 筛选条件同时作用于：数据概览、反馈列表、AI分析
- 三个模块根据 FilterStore 中的筛选条件获取数据

## 3. 验收标准

- [ ] 左侧导航只有两个一级菜单：工作台、用户反馈
- [ ] 用户反馈下有二级菜单：反馈列表、数据仪表盘、AI智能分析
- [ ] 二级菜单选中时才高亮
- [ ] 右侧内容区显示：筛选条件、反馈列表、数据仪表盘、AI智能分析 四个模块（从上到下）
- [ ] 每个模块有明确的标题名称
- [ ] 修改筛选条件，三个模块数据同步更新
- [ ] 点击二级菜单，页面滚动到对应模块
