# 项目技术配置

## 一、技术栈

| 层级 | 技术选型 | 说明 |
|------|----------|------|
| **前端** | React + TypeScript + Tailwind CSS | SPA 单页应用 |
| **后端** | Python + FastAPI | 高性能 API 框架 |
| **数据库** | MySQL | 适配阿里云宝塔面板 |
| **AI API** | Kimi API / MiniMax API | 已提供密钥 |
| **图表** | ECharts | 可视化仪表盘 |
| **部署** | 宝塔面板 + Docker | 阿里云服务器环境 |

## 二、已确认的决策

| 决策项 | 结论 | 说明 |
|--------|------|------|
| **用户系统** | 暂不接入 | 后期再考虑 |
| **权限管理** | 不需要 | 内部工具，暂不设计 |
| **系统集成** | 无需集成 | 独立运行 |
| **数据规模** | ~100条 | 无敏感数据 |
| **交付标准** | 部署上线 | 需要部署文档 |
| **部署目标** | 阿里云宝塔 | 云服务器 |
| **API 密钥** | 环境变量 | .env 文件管理 |

## 二、部署环境

| 项目 | 配置 |
|------|------|
| 服务器 | 阿里云云服务器 |
| 管理面板 | 宝塔面板 |
| 数据库 | MySQL（宝塔安装） |
| Python | 3.10+ |
| Node.js | 18+ |
| 容器 | Docker（可选） |

## 三、简化决策（已确认）

### 当前范围
- ✅ 反馈列表与管理
- ✅ 数据统计仪表盘
- ✅ AI 智能分析
- ❌ 用户系统（暂不接入）
- ❌ 权限管理（不需要）
- ❌ 系统集成（独立运行）

### 原因
- 内部工具，数据量小（~100条）
- 无敏感数据
- 快速交付优先
